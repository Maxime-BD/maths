La vie est belle
Le soleil brille
Les oiseaux chantent
Les fleurs s'épanouissent doucement
Les rires résonnent joyeusement
Les enfants jouent
Les vagues caressent doucement
Le rivage doré
Les étoiles scintillent
Dans le ciel étoilé
Plein de mystères
Les amis se retrouvent
Autour d'une table conviviale
Pour partager un bon
Les livres ouvrent l'esprit
Sur des mondes imaginaires
Et des connaissances infinies
Les pas dansent légèrement
Au rythme d'une mélodie
Les nuages flottent paisiblement
